== Url Shortener Api

A simple api to create short urls with custom domain. 

GET /short_urls/:url (Create shortened url)

Input

url: Input URL to shorten
Output (JSON)


url: Shortened URL

GET /short_urls/:id

Input

Input hash to get original URL

Output (JSON)

url: Original URL


GET /:short_url (redirect)


Input short url => Redirect to original URL
